      시스템 개선 필요사항 및 기능 제안

      1. 보고서 구조와 편집 기능

      • 기존 HTML 보고서는 커스터마이징 한계가 큼.
      • 슬라이드 형태 구성 또는 PPT/이미지 삽입 방식 검토.
      • 사례 내용 선택 및 사용자 맞춤형 입력 요구.
      • 보고서 목차 구성 일부 변경 필요.

      2. 디자인/테마 커스터마이징

      • 색상, 폰트 등 테마 선택 기능 제안.
      • 전체 테마 적용 방식(개별 슬라이드 변경 아님).
      • 슬라이드 템플릿 2분할, 3분할 등 선택 방식.
      행정사항
      유관기관 한국장애인고용공단 경기서부지사, 행복누리, 부산시 장애인 종합복지관
